# ZLX Security
===============

Security class used for encrypting, decrypting, and hashing

## Installation

In your ```composer.json``` file include:

```
"require": {
    "aledefreitas/zlx_security": "*"
}
```

Or you can require it with composer:

```
composer require aledefreitas/zlx_security
```

## Requirements

- PHP 5.6 +
- EXT-OPENSSL
- EXT-MBSTRING
